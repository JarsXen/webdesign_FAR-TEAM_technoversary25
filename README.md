EcoDigital Nusantara adalah platform web modern yang dirancang untuk meningkatkan kesadaran lingkungan melalui pendekatan digital yang interaktif. Website ini menggabungkan edukasi, aksi nyata (seperti bank sampah), dan kecerdasan buatan (EcoBot) untuk membantu pengguna memulai gaya hidup zero-waste.

### Fitur Utama:
1.  **EcoBot Assistant (AI Powered):** Chatbot pintar berbasis Google Gemini untuk konsultasi masalah lingkungan.
2.  **Bank Sampah Digital:** Kalkulator estimasi nilai tukar sampah daur ulang.
3.  **Edukasi Interaktif:** Artikel dan tips praktis seputar pelestarian alam.
4.  **Galeri Dokumentasi:** Kumpulan foto aksi nyata pelestarian lingkungan.
5.  **Desain Responsif & Modern:** Menggunakan gaya Glassmorphism dan animasi halus untuk pengalaman pengguna terbaik.

## ğŸš€ Cara Menjalankan Project

Pastikan Anda sudah menginstal **Node.js** di komputer Anda.

1.  **Install Dependensi**
    Buka terminal di folder project dan jalankan:
    ```bash
    npm install
    ```

2.  **Konfigurasi API Key**
    Buat file bernama `.env` di folder root (sejajar dengan package.json), lalu isi dengan API Key Google Gemini Anda:
    ```env
    VITE_API_KEY=isi_api_key_gemini_anda_disini
    ```
    *(API Key bisa didapatkan gratis di aistudio.google.com)*

3.  **Jalankan Server Development**
    ```bash
    npm run dev
    ```
    Buka browser dan akses `http://localhost:5173`
